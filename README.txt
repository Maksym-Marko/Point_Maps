		PointMaps - Программа для путешественников.
 
ИДЕЯ:
С помощью PointMaps, можно загружать карту, делать на ней пометки необычных находок и отправлять их друзьям.

ХОД ВЫПОЛНЕНИЯ:
29.08.16 – с помощью курсора мыши, на карте можно ставить точки.
19.09.16 - можно ставить неограниченное количество точек.
19.09.16 - можно добавлять описание точки
20.09.16 - добавлена возможность перевода массива данных в JSON-формат.
